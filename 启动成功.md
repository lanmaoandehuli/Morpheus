## 🔧 解决闪退问题

### 问题原因
1. ❌ 缺少 `pydantic_settings` 模块
2. ❌ Python 3.14 + 多 worker 模式内存不足

### 解决方案
1. ✅ 安装了 `pydantic-settings`
2. ✅ 改为单 worker 模式 (`API_WORKERS=1`)
3. ✅ 清理了占用端口的旧进程

## 🚀 如何停止服务

### 方式 1: 关闭命令行窗口
直接关闭运行后端的 PowerShell 窗口

### 方式 2: 命令行停止
```powershell
# 停止所有 Python 进程
Stop-Process -Name "python" -Force
```

### 方式 3: 使用任务管理器
1. Ctrl+Shift+Esc 打开任务管理器
2. 找到 `python.exe` 或 `uvicorn`
3. 右键 → 结束任务

## 📖 快速开始

### 1. 创建第一个项目

访问 http://localhost:3000

1. 点击 **"创建项目"**
2. 填写：
   ```
   项目名称：末世修仙录
   类型：玄幻
   风格：热血升级
   目标字数：2000000
   世界观：主角重生到末世，获得修仙系统
   ```
3. 点击 **"创建"**

### 2. 创作第一章

1. 进入项目详情
2. 点击 **"创建章节"**
3. 输入：**"第 1 章 重生"**
4. 点击 **"生成章节计划"**
5. 审查计划（AI 生成 5-10 个节拍）
6. 点击 **"生成草稿"**
7. 实时观看 AI 创作（流式输出）
8. 审查、润色、定稿

## 🧠 核心功能

### 三层记忆系统
- **L1 Identity**: 世界观、主角设定
- **L2 Entities**: 人物、地点、物品
- **L3 Events**: 关键情节、时间线

### 四个 Agent
- **Director**: 规划 + 初稿
- **Setter**: 设定审查
- **Stylist**: 文笔润色
- **Arbiter**: 最终定稿

### 实时流式生成
- SSE 推送创作进度
- 实时显示 AI 写作过程
- 支持批量生成
