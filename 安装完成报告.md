## 📖 快速开始教程

### 第 1 步：创建第一个项目

1. 访问 http://localhost:3000
2. 点击 **"创建项目"**
3. 填写信息：
   ```
   项目名称：末世修仙录
   类型：玄幻
   风格：热血升级
   目标字数：2000000
   世界观：主角重生到末世，获得修仙系统
   ```

4. 点击 **"创建"**

1. 进入项目详情页
2. 编辑 **"世界观设定"**
3. 添加 **"主角设定"**

1. 点击 **"创建章节"**
2. 输入章节名：**"第 1 章 重生"**

4. 审查计划（AI 会生成 5-10 个节拍）
5. 点击 **"生成草稿"**
6. 实时观看 AI 创作过程（流式输出）

### 第 4 步：继续创作

1. 创建第 2 章、第 3 章...

4. 定期查看 **"知识图谱"**

## 🧠 三层记忆系统

Morpheus 会自动维护：

### L1 Identity（核心设定）
- 世界观规则
- 主角基本信息
- 故事风格

### L2 Entities（实体信息）
- 人物档案（性格、外貌、关系）
- 地点设定（特征、位置）
- 重要物品（功能、归属）

### L3 Events（事件记录）
- 关键情节
- 时间线
- 因果关系

**优势**：
- ✅ 自动检索相关记忆
- ✅ 保持前后一致性
- ✅ 避免吃书错误

## 🔧 常见问题

### Q1: 前端页面打不开？
**A**: 
1. 确保前后端都启动了
2. 检查端口是否被占用
3. 访问 http://localhost:3000

### Q2: 生成一直是降级文本？
**A**:
1. 访问 http://127.0.0.1:8000/api/runtime/llm
2. 确认 `remote_ready: true`
3. 如为 false，检查：
   - API Key 是否正确
   - 网络连接是否正常
   - 账户是否有余额

### Q3: 后端启动失败？
**A**:
```powershell
# 查看详细错误
cd backend
python -m uvicorn api.main:app --reload

# 检查端口占用
netstat -ano | findstr :8000
```

### Q4: 向量检索报错？
**A**:
```powershell
# 删除向量数据库重建
cd D:\openclaw\.openclaw\Morpheus
Remove-Item -Recurse -Force data\vectors\*
# 重启后端
```
