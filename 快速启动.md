2. ✅ 后端配置已创建：`backend\.env`

## ⚠️ 下一步操作（按顺序）

### 1️⃣ 获取 DeepSeek API Key

访问：https://platform.deepseek.com/
1. 注册/登录
2. 进入"API 管理"
3. 创建 API Key
4. 复制 Key（格式：`sk-xxxxxxxx`）

### 2️⃣ 配置 API Key

用记事本打开：`D:\openclaw\.openclaw\Morpheus\backend\.env`

找到这一行：
```
DEEPSEEK_API_KEY=sk-your-deepseek-api-key-here
```

改为你的真实 Key：
```
DEEPSEEK_API_KEY=sk-你的真实 APIKey
```

保存并关闭。

### 3️⃣ 安装依赖

**打开 PowerShell，运行**:
```powershell
# 安装后端依赖
cd D:\openclaw\.openclaw\Morpheus\backend
pip install fastapi uvicorn python-dotenv lancedb openai pydantic

# 安装前端依赖
cd ..\frontend
npm install
```

### 4️⃣ 启动服务

**方式 1: 双击启动（推荐）**
- 双击 `D:\openclaw\.openclaw\Morpheus\start.bat`

**方式 2: 手动启动**
```powershell
# 终端 1 - 启动后端
cd D:\openclaw\.openclaw\Morpheus\backend
python -m uvicorn api.main:app --host 127.0.0.1 --port 8000 --workers 2

# 终端 2 - 启动前端
cd D:\openclaw\.openclaw\Morpheus\frontend
npm run dev
```

## 🔧 常见问题

### Q: 看不到前端页面？
A: 确保前后端都启动了，访问 http://localhost:3000

### Q: 生成一直是降级文本？
A: 
1. 检查 API Key 是否正确
2. 访问 http://127.0.0.1:8000/api/runtime/llm
3. 确认显示 `"remote_ready": true`

### Q: 安装依赖失败？
A: 
```powershell
# 升级 pip
python -m pip install --upgrade pip

# 重新安装
pip install -r requirements.txt
```
